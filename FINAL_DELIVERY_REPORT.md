## 🚀 使用流程

### 1. 录音流程
```
打开应用 → 点击录音按钮 → 开始录音
         ↓
    实时波形可视化 + 粒子效果
         ↓
    点击停止 → 自动语音识别
         ↓
    选择情绪 → AI 自动分析
         ↓
    保存到数据库 + 显示在时间轴
```

### 2. AI 分析流程
```
录音完成 → 转写文本
         ↓
    并行调用 3 个 AI 接口
         ↓
    生成摘要 + 情绪分析 + 标签
         ↓
    显示在播放器页面
```

### 3. 查看趋势流程
```
主页 → 点击趋势图标 → 趋势分析页面
         ↓
    选择时间范围（7/30/90 天）
         ↓
    查看折线图 + 分布图 + 洞察
```

### 4. AI 对话流程
```
主页 → 点击 AI 图标 → 对话页面
         ↓
    输入问题（如："我最近心情怎么样？"）
         ↓
    AI 基于历史录音回答
         ↓
    继续对话或清空历史
```

## 💡 使用建议

### 最佳实践
1. **录音长度**: 建议 30 秒以上，AI 分析更准确
