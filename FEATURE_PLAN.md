# VoiceMemory 完整功能规划

## 🎨 UI/UX 设计方向

### 视觉风格
- **Glassmorphism** (毛玻璃效果)
- **3D 卡片动画** (深度感、阴影)
- **流体动画** (Lottie/自定义 Canvas)
- **粒子效果** (录音时的动态粒子)
- **渐变色彩** (情绪主题色渐变)
- **微交互** (按钮点击、滑动反馈)

### 核心页面

#### 1. 录音页面 (主页)
- 3D 悬浮录音按钮 (带阴影和光晕)
- 实时波形可视化 (多层次、渐变色)
- 粒子系统 (声音强度驱动)
- 情绪色彩氛围灯效
- 录音时长 + 剩余时间
- 暂停/继续/停止控制
- 实时转写预览

#### 2. 时间轴页面
- 垂直时间线布局
- 3D 卡片堆叠效果
- 情绪色彩标签
- 滑动删除/编辑
- 下拉刷新动画
- 日期分组
- 搜索栏 (模糊搜索)

#### 3. 日历热力图
- 月视图 + 年视图
- 热力色阶 (录音频率)
- 点击查看当日录音
- 情绪分布饼图
- 统计数据卡片

#### 4. 播放器页面
- 全屏沉浸式播放
- 歌词式滚动文本
- 波形进度条
- 播放速度控制
- 循环/单曲播放
- 分享/导出功能

#### 5. 时光胶囊
- 锁定动画效果
- 倒计时显示
- 解锁特效
- 胶囊列表

#### 6. 设置页面
- 主题切换 (深色/浅色/自动)
- 语音识别设置
- 存储管理
- 数据导出/导入
- 关于页面
