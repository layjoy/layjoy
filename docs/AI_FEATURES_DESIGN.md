# VoiceMemory AI 功能设计方案

## 🤖 AI 大模型功能规划

### 核心 AI 功能

#### 1. 智能摘要生成
**场景**: 长录音自动生成摘要
- 输入：转写文本（500+ 字）
- 输出：3-5 句话的核心摘要
- 用途：快速浏览历史记录

#### 2. 情绪深度分析
**场景**: 超越关键词的情感理解
- 输入：转写文本
- 输出：情绪类型 + 强度 + 原因分析
- 示例：
  ```
  情绪：焦虑 (强度: 7/10)
  原因：工作压力大，担心项目进度
  建议：适当休息，分解任务
  ```

#### 3. 主题标签提取
**场景**: 自动分类和标签
- 输入：转写文本
- 输出：3-5 个主题标签
- 示例：#工作 #压力 #健康 #人际关系

#### 4. 智能问答
**场景**: 与历史记录对话
- 输入：用户问题
- 输出：基于历史录音的回答
- 示例：
  - "我上周心情怎么样？"
  - "我最近在担心什么？"
  - "给我一些建议"

#### 5. 趋势分析报告
**场景**: 周/月情绪趋势
- 输入：一段时间的所有录音
- 输出：情绪变化曲线 + 分析报告
- 示例：
  ```
  本周情绪趋势：
  - 整体偏向焦虑（60%）
  - 周三情绪最低
  - 主要压力源：工作
  - 建议：增加运动和社交
  ```

#### 6. 写作助手
**场景**: 将录音转为文章
- 输入：转写文本
- 输出：结构化文章（标题+段落）
- 用途：日记、博客、社交分享

#### 7. 语音日记教练
**场景**: AI 陪伴和引导
- 输入：用户录音
- 输出：鼓励、建议、提问
- 示例：
  ```
  "听起来你今天很开心！是什么让你这么高兴？"
  "我注意到你最近经常提到工作压力，要不要聊聊？"
  ```

#### 8. 关键词云图
**场景**: 可视化高频词汇
- 输入：所有录音文本
- 输出：词云图 + 频率统计
- 用途：了解自己最关注的话题

## 🏗️ 技术实现方案

### 模型选择

#### 方案 A: OpenAI API
```kotlin
模型: GPT-4 Turbo
优点: 效果最好，功能全面
缺点: 需要付费，有网络延迟
成本: ~$0.01/1K tokens
```

#### 方案 B: 国产大模型
```kotlin
选项:
1. 讯飞星火 - 已有账号，可复用
2. 智谱 GLM-4 - 性价比高
3. 百度文心 - 免费额度大
4. 阿里通义千问 - 速度快

推荐: 讯飞星火（与语音识别同一账号）
```

#### 方案 C: 本地模型
```kotlin
模型: Gemini Nano (Android 14+)
优点: 完全离线，隐私保护
缺点: 功能受限，需要高端设备
```

### 推荐方案
**混合方案**: 讯飞星火（主） + Gemini Nano（辅）
- 在线功能：摘要、深度分析、问答
- 离线功能：情绪识别、关键词提取

## 💰 成本估算

### 讯飞星火 API
- 免费额度：200万 tokens/月
- 付费价格：¥0.018/千tokens
- 预估成本：
  - 每条录音分析：~500 tokens
  - 每月 100 条录音：¥0.9
  - 年成本：~¥11

### 数据隐私
- 所有数据加密传输
- 不存储用户原始录音到云端
- 仅发送转写文本
- 用户可选择关闭 AI 功能
